oddfruit bitmap logo assets
===========================

These files use the supplied Option 5 Open Orbit bitmap reference as the source of truth.
No SVG logo files are included in this pack. A final SVG/vector master can be created later.

Recommended website files:
- oddfruit-logo-horizontal-1200.png: large horizontal logo
- oddfruit-logo-horizontal-600.png: standard header logo
- oddfruit-mark-open-orbit-512.png: standalone mark
- favicon-32x32.png and favicon-16x16.png: site favicons at the web root
- apple-touch-icon.png: iOS home screen icon at the web root

Variants:
- horizontal logo, light background
- horizontal logo, dark background
- standalone mark
- standalone mark, dark background
- round badge
- rectangular badge

Notes:
The current source is bitmap, so use the largest practical PNG size when designing new materials.
