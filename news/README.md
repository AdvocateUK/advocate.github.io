# Adding a new oddfruit news post

1. Duplicate one of the existing `.html` files in this folder, or start from `_template.html`.
2. Rename it with a simple slug, for example `inr-mate-v13-app-intents.html`.
3. Update the page title, meta description, canonical URL, date, heading and article text.
4. Add a new card near the top of `news/index.html`.
5. Commit and push to GitHub Pages.

Keep images in `assets/` and reference them from news posts with `../assets/...`.
